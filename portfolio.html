Match Game

A Javascript program that, with JQuery, simulates a match-two game! Click on a square to reveal a color, then another. If the colors match, they fade away. If they don't, the colors will wait for a second, then flip back. I'm kinda proud of it.

Written in HTML5, CSS, Javascript, and JQuery.

Listmaker

This Javascript program, enhanced with JQuery, tracks several items at once. Enter something to add it to the list. Click once to mark it out, and click again to make it disappear from the list.

Written in Javascript and JQuery,
with HTML5 and CSS edits.
